## What happens when you save this and paste it in

Open Claude Code or Claude Desktop and say:

> Read my INTAKE.md and run the brand-in-a-box skill.

In about 90 seconds, Claude writes you:

- A `brand/` folder with positioning, voice DNA, ICP, colors, typography, tokens.json
- A `colors_and_type.css` file you can drop into any web project
- A personalised `SKILL.md` so any Claude session writes and designs in your brand
- A `DESIGN.md` you can drop into Claude Design to skin your projects automatically
- One sample artifact per format you ticked in question 8

You will not have a finished brand.

You will have a starter that beats 90% of what most founders ship in their first year.
