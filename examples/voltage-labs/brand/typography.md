# Voltage Labs — Typography

Generated by brand-in-a-box. Family default for Cinematic Dark.

## Families

Heading: Space Grotesk (700 / 600). Geometric sans, slightly mechanical.

Body: Inter (400 / 500).

Mono: JetBrains Mono (500). For eyebrows, code, version strings, technical labels.

All three free on Google Fonts.

## Hard rules

Display scale (120px) is permitted on landing-page heroes. Cinematic Dark earns size — the void around the type is what makes it cinematic.

Eyebrows are mono, not sans. The mono creates a "system label" feel that signals technical credibility.

Hero line-height is tight (1.05). On 80px+ headlines, anything looser feels casual.

Tracking on hero is -0.02em. On display (120px+) it goes to -0.04em.

## Sizing notes

Display (120px) on landing hero only. Reserve for the one big idea per page.

Hero on 1080×1350 carousel cover: 64-80px.

Hero on 1584×396 banner: 64-72px.

Body never below 16px shipped. Captions 13px minimum.
