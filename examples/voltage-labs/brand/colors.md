# Voltage Labs — Colors

Generated by brand-in-a-box. Cinematic Dark family.

Pure black canvas. Electric cyan as the only accent. Everything else lives in the void-to-pearl gradient.

| Token | Hex | Usage |
|---|---|---|
| Cyan | #00d4ff | Primary accent. CTAs, eyebrows, technical labels, code highlights. Glows on near-black. Cap at 6% of any frame. |
| Pearl | #f5f5f7 | Primary text on dark. Slightly off-white — pure white is too harsh on void. |
| Iron | #2c2c30 | Borders, dividers, subtle UI separators. |
| Void | #000000 | Default canvas. Pure black. The brand is unapologetic about it. |
| Charcoal | #0a0a0c | Card and elevated surfaces. Sits 1 step above void. |
| Graphite | #16161a | Higher elevation, hover states. |
| Smoke | #6e6e73 | Secondary text. |
| Fog | #98989d | Tertiary text, muted labels. |

The 6% rule on cyan keeps the glow legible. Cyan everywhere = Cyan nowhere.

Use cyan on:

- The CTA button (one per frame).
- One technical highlight per section (a number, a label, a metric).
- The eyebrow on a featured section.
- Code-block keywords if the artifact contains code.

Never use cyan on body paragraphs. The contrast is too high for sustained reading.

## Elevation through dark layering

The brand has no shadows. Elevation is the gradient void → charcoal → graphite. Three steps, each ~1% lighter.

A card lifts because its background is charcoal against void. A modal lifts because its background is graphite against charcoal. The eye reads the difference instinctively.
