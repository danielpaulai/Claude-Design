## How this differs from the Purely Personal example

| Dimension | Purely Personal (Editorial Minimal) | Marigold & Oak (Warm Editorial) |
|---|---|---|
| Heading font | Rethink Sans (sans-serif) | Crimson Pro (serif) |
| Body size | 16px / 1.5 line-height | 17px / 1.65 line-height |
| Card corners | 0px (editorial) | 8px (softer) |
| Card borders | 2-3px solid near-black | 1px solid warm-gray |
| Accent cap | 5% of frame | 8% of frame |
| Sentence rhythm | One per paragraph (avg 9 words) | Flowing (avg 18 words) |
| Imagery | Founder portraits only, near-black backdrop | Naturalistic, golden-hour, hands and texture |
| Anti-slop emphasis | No SaaS gradients, no Poppins | No "boss babe" wellness tropes, no script fonts |

Same generator. Same SKILL. Different family. Two coherent systems.

## How to use this folder

The same three lanes as any brand-in-a-box output:

**Lane 1: Inside Claude Code or Claude Desktop.**
Drop this folder into a session. The SKILL activates. Output stays in voice and on-brand.

**Lane 2: Inside Claude Design.**
Open `DESIGN.md`. Upload to claude.ai/design. Every project from then on inherits this design.

**Lane 3: As source for any other tool.**
`brand/tokens.json` is the source of truth. Regenerate Figma, Canva, or CSS from it.
