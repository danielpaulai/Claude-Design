# One-pager UI Kit — A4 @ 300dpi (2480×3508)

Letter-press inspired offer one-pager. Single printable page. Off-white paper, 2px black rules, giant sans-serif numerals, red accent.

## Components

- `OnePagerFrame.jsx` — full A4 canvas scaled to fit viewport
- `OfferHero.jsx` — eyebrow + huge headline + subtitle
- `ProofRow.jsx` — three stat numerals in a row (e.g. "12h → 2h")
- `InclusionsList.jsx` — black rule + "You will own:" + numbered bullets
- `CTAFooter.jsx` — book-a-call block with byline

Open `index.html` for a full offer sheet mock.
