# Carousel UI Kit — 1080×1350

LinkedIn document carousel format. 8–12 slides per post: cover → hook → body → payoff → CTA. Exported as PDF.

## Components

- `CarouselFrame.jsx` — fixed 1080×1350 canvas with scale-to-fit
- `CoverSlide.jsx` — hero title + byline
- `HookSlide.jsx` — big statement frame
- `StepSlide.jsx` — numbered step with prompt callout
- `CTASlide.jsx` — final "Comment X" slide
- `SlideFooter.jsx` — Daniel byline + Repost pill (persistent bottom module)

## Canvas

1080×1350 (vertical 4:5), 80px outer margin. Off-white paper canvas. Footer module ~120px tall at the bottom of every slide except the cover.

Open `index.html` for a navigable 10-slide mock carousel.
