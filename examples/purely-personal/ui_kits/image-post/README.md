# Image-post UI Kit — 1080×1350

Single big-idea image post (JPEG). Two canonical patterns:

- **Portrait + headline** — moody founder portrait on near-black, LinkedIn-style author card inset, bold white headline across bottom (see `reference/image-post-example.jpeg`).
- **Text-only statement** — no imagery. Large Rethink Sans headline on paper canvas.

## Components

- `ImagePostFrame.jsx` — 1080×1350 canvas
- `PortraitStatement.jsx` — full-bleed portrait + author card + bottom headline
- `TextStatement.jsx` — paper canvas + giant statement
- `LinkedInAuthorCard.jsx` — avatar + name + verified + subtitle, in a capsule

Open `index.html` to toggle between variants.
