# Purely Personal — Design Skill

Generated by brand-in-a-box.

Read `README.md` first. It indexes everything in this skill.

## Hard rules you must not violate

Voice (extracted from your samples in INTAKE Q7):

- One sentence per paragraph (avg 9 words).
- Plain English. No corporate jargon.
- No em dashes. Ever.
- No hashtags on LinkedIn.
- No emojis in professional copy.
- "I" + "you" pronouns. Never "we" except in positioning copy.
- Sentence case for everything except proper nouns.

Visual:

- Rethink Sans for headings. Inter for body. Never Poppins.
- Red `#e90d41` capped at 5% of any frame. Never on paragraphs.
- 2-3px solid near-black borders on cards. 0px corner radius for editorial frames.
- No stock photography. No SaaS gradients. No glassmorphism.

Banned phrases (from `brand/voice-dna.md`):

`growth hack` · `10x` · `guru` · `unlock` · `leverage` · `hustle` · `synergy` · `disrupt` · `game-changer` · `secret sauce` · `follow for more`

## When invoked for a visual artifact

1. Load `colors_and_type.css`.
2. Use the canvas size from `brand/tokens.json` matching the requested format.
3. Generate static HTML the user can open directly.
4. Pass anti-slop rules in `brand/anti-slop.md`.
5. Use real example copy from `brand/positioning.md` (avoid lorem ipsum).

## When invoked for copy

1. Read `brand/voice-dna.md` for hooks, rhythm, banned phrases.
2. Pick a hook mechanism. Three available, ranked: contrarian observation, transformation, category reframe.
3. Output. No banner sentences, no em dashes, no hashtags.

## When invoked without specific guidance

Ask:

- What format? (carousel, cheatsheet, image post, one-pager)
- What hook mechanism? (contrarian, transformation, category reframe)
- CTA keyword? (e.g. "AI", "TEAM")

## Default outputs

Carousel: 1080×1350 PDF, 8-12 slides. Cover + steps/lessons + CTA.

Cheatsheet: 1080×1350 JPEG, single dense frame, optional pastel category bands.

Image post: 1080×1350 JPEG, one big idea, founder portrait or full-bleed type.

One-pager: A4 @ 300dpi, letter-press editorial, hero + offer top, proof + CTA bottom.
