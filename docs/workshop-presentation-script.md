# Backup plans (when things go wrong)

| Failure | Backup |
|---|---|
| Wifi dies mid-demo | Switch to pre-rendered screenshots on Desktop. Don't apologise. Keep momentum. |
| Claude Design takes >2 min to render | Talk through what's happening (the inference, the soul check, the rules) — turn the wait into education |
| Volunteer's brand generates with weak voice | "The samples weren't strong enough — that's the prompt 05 fix." Move on. |
| Master Check returns wrong values | DESIGN.md isn't loaded. Re-upload live. Audience sees the failure mode + recovery. Don't hide it. |
